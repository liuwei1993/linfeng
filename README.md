#### 1.介绍

林风婚恋交友uniapp**小程序/H5版本**是前后端分离的婚恋交友SNS项目。 项目基于以下技术栈开发：SpringBoot、MybatisPlus、Shiro、Quartz、jwt、netty、websocket、RabbitMQ、Redis、Vue、Uniapp、MySQL。
