## 🚀 推荐使用方式

### 新手 (5 分钟快速体验)

```bash
cd /home/liuwei/codes/linfeng-love

# 1. 启动应用
docker-compose up -d

# 2. 等待 30 秒后访问
# 后端 API: http://localhost:8080
# Swagger: http://localhost:8080/swagger-ui.html
# RabbitMQ: http://localhost:15672
```

### 开发者 (1-2 小时深入学习)

```bash
# 1. 阅读快速开始指南
cat QUICK_START.md

# 2. 阅读系统架构文档
cat TECHNICAL_ARCHITECTURE.md

# 3. 本地开发环境搭建
./setup-environment.sh dev

# 4. 启动并开发
cd linfeng-love-backend-open
mvn spring-boot:run
```

### 运维人员 (完整的部署知识)

```bash
# 1. 阅读部署指南
cat DEPLOYMENT_GUIDE.md

# 2. 生产环境部署
./deploy.sh deploy prod
./deploy.sh start

# 3. 配置 Nginx
cp nginx.conf /etc/nginx/

# 4. 监控和维护
./deploy.sh status
./deploy.sh logs
```
